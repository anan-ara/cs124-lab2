Clicking on the menu icon

This shows the result of clicking on the menu icon to show more actions. Note that not only do we have a "Rename" option here, but we also have the options to change the priority level of the todo item, as well as to delete the item entirely. Renaming, changing the priority level, and deleting the item are all tasks that we would rather not happen accidentally, so putting them under a menu ensures that the user will not accidentlaly take these actions.
